== Installation ==

= Installation =

* Important: You need to have installed Visual Composer;
* Upload the plugin to the wp-content/plugins folder;
* Activate the plugin;
* You must need update the your .htaccess, because this plugin create a new custom post type and a personality single;

= How to use: =

* After activated you will see a new tab "Showcases".
* For add a new Showcase, you need to go "Showcases -> Add New".
* Inside of the new post, you have a few fields, like : Title, Content and Featured Image. 
* Below content you go to see a featured links, where you can add your custom links.
* For add the carrousel, go in your page and click in "Add new element" and found per "Showcases".
* With your new element "Showcases" set the slide as you want.
